# THE SEMANTICS



                   This essentially means there is a new set of tags that works in enhancing your HTML code, helping make it increasingly meaningful. 
                    Conventional developers use a lot of tag div. Now, they can easily develop using the new tags that include nav, header, and footer. 
                    This makes HTML more productive. When it comes to the classification of different webpage parts, HTML5 provides all the necessary 
                    semantic elements.Some of the popular semantic elements are article, aside, details, figcaption, figure, footer, header, main, mark,
                    nav, section, summary, time etc. which are very useful elements of HTML5.


# VEDIO MULTIMEDIA



      HTML5 has the ability to integrate video and audio smoothly into browsers, without the need for Silverlight or Flash.
    Initially, there was no seamless way for videos to be incorporated into website pages. This resulted in a need for video player development.
    While these players functioned well, the need for external plug-ins meant that you had to go through the annoying process of downloading
    them. Now, HTML5 does the work for you. Videos run smoothly and users get a good browsing experience without any delays.


# 3D GRAPHICS AND SPECIAL EFFECTS



                    In the past, developers had to use JavaScript or Flash/CSS in the fabrication of animation for website visual effects.
                    HTML5 comes with new tags such as canvas that give developers the ability to provide great visual effects which are similar to those produced
                    by Flash.  This explains why many experts feel it might be the right time to say goodbye to Flash. HTML5 also provides other amazing features
                    such as WebGL and 2D Canvas, geared towards web graphics. This just gives HTML5 increased authority in this sphere.


# GAME DEVELOPENT



       When using HTML5 for game development, ongoing maintenance becomes easy owing to a single code base. The configuration
        of new updates takes place automatically. Resubmitting the game to different app stores is not required, as users don’t store HTML5 games on
        their devices.Since HTML5 games offer increased access to users, they are capable of reaching wide audiences. This gives developers the
        means to monetize their games by offering in-game advertisements that promote other apps or websites. In addition, sharing HTML5 games 
        through featured links on gaming-based websites is easy.

# FLASH REPLACEMENT



       Mobile devices, without a doubt, have won the computer vs. mobile devices war. They have taken over the world, and an increasing
                 number of people are starting to use them to perform internet-based activities. HTML5 is among the most favored tools by developers who brace 
                 for a future full of mobile devices. It has the capability of developing modern applications as well as mobile-friendly sites. Adobe is already 
                 indicating the death of Flash and developers will have to depend on HTML5 to handle web app development in the near future.


ADVANTAGES OF HTML5 OVER HTML

  • HTML5 can easily detect a user’s location and then use it to provide tailor-made search results.
  • It can even tag social media updates. In addition, it does not limit to a single provider’s browser tool or application program interface (API).
  • HTML5 has made way for better controls when it comes to dealing with textual inputs in search boxes and elsewhere.
  • Validating data, sending data via email, and interaction of textual inputs with other page elements has become streamlined.
  • It is safe to say that HTML5 has simplified adding discussion boards, drag-and-drop tools, wikis, chat platforms, and a host of other elements into just about any website.
  • As HTML5 adoption continues to gain traction, websites will load faster. This results in decreased bandwidth usage, which, in turn, leads to longer battery life for mobile devices.
